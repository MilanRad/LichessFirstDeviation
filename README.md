# LichessFirstDeviation
Calculates the change in winrate sorted by which player is the first to create a brand new game not found in the database with their latest move.

Repo is missing the database of lichess games as those files are too large. They can be downloaded at: https://database.lichess.org/
My choice was the 2017 February data set: https://database.lichess.org/standard/lichess_db_standard_rated_2017-02.pgn.zst
Compressed size: 1.7GB, decompressed size: 9GB

Running time on 16GB RAM, AMD Ryzen 5 2600 Six-core processor, Radeon RX 580, with SSD, was around 5 minutes.
